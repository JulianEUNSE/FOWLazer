# FOWLazer

A Phaser-based browser game that I am creating as a side hobby.

# Getting started

1. Clone the repository \n
``` git clone https://github.com/JulianEUNSE/FOWLazer.git ``` \n

2. Install Dependencies \n
``` npm install ``` \n
``` npm install phaser ``` \n

3. Run the development server \n
``` npm run dev ``` \n

From here, you can follow the link to open the game on your own localhost.
